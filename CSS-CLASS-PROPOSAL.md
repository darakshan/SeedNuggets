# Proposal: CSS class names to replace embedded styles

All HTML is generated by `src/build.py`. The boilerplate stylesheet is the same in every doc; only inline `style="..."` varies. This document lists every embedded style and proposes a single class (or composite) to move into the stylesheet. **No code changes yet** ‚Äî review first.

---

## 1. Nugget pages (001-caloric.html, etc.)

Generated in `build_nugget()`, `text_to_html()`, `script_to_html()`.

| Current inline style | Where used | Proposed class | Notes |
|---------------------|------------|----------------|-------|
| `font-style:italic` (with `.dim`) | TBD layer placeholder, "Script not yet written" | **`.placeholder`** or **`.tbd`** | Reuse for any "not yet written" / italic dim text. |
| `margin-bottom:.5rem` | "Related seeds" label above related cards | **`.related-label`** | Already has `.mono.small.warm`; add this class for margin. |
| Script: `font-family:'DM Mono',monospace; font-size:.62rem; letter-spacing:.15em; text-transform:uppercase; color:var(--warm); margin-top:1.5rem` | Script direction lines (OPEN, CUT TO, etc.) | **`.script-direction`** | One class for script directions. |
| Script: `font-size:1.3rem; font-style:italic; color:var(--accent); margin-top:1.5rem` | Script punch lines ("What does‚Ä¶", "What if‚Ä¶") | **`.script-punch`** | One class for the closing question/lines. |
| Script: `margin-bottom:.6rem` | Script spoken lines (body) | **`.script-line`** | Tight spacing between script lines. |

**Suggested CSS additions (nugget):**

- `.placeholder` or `.tbd` ‚Äî `font-style: italic;` (color already from `.dim`).
- `.related-label` ‚Äî `margin-bottom: .5rem` (use with existing `.mono.small.warm`).
- `.script-direction` ‚Äî mono, small caps, warm, top margin.
- `.script-punch` ‚Äî larger, italic, accent, top margin.
- `.script-line` ‚Äî `margin-bottom: .6rem`.

## 7. Summary: new classes to add to the stylesheet

**Nugget / script**

- `.placeholder` ‚Äî italic (pair with `.dim` where needed).
- `.related-label` ‚Äî `margin-bottom: .5rem`.
- `.script-direction` ‚Äî script direction line (mono, small caps, warm, margin-top).
- `.script-punch` ‚Äî script closing question (larger, italic, accent, margin-top).
- `.script-line` ‚Äî `margin-bottom: .6rem` for script body lines.

**Index**

- `.hero` ‚Äî hero block padding.
- `.hero-notice` ‚Äî block, margin (e.g. "Working archive‚Ä¶").
- `.hero-tagline` ‚Äî tagline size, color, max-width, line-height.
- `.hero-stats` ‚Äî flex stats row with top border.
- `.hero-stat-label` ‚Äî label under stat number.
- `.seed-list-section` ‚Äî padding for seed list block.
- `.section-head` ‚Äî flex row with bottom border (and optional link).
- `.link-mono-small` ‚Äî nav-style small caps link (dim).
- `.seed-list-more-wrap` ‚Äî padding around "view all" link.
- `.link-mono-accent` ‚Äî small caps link (accent color).
- `.about-block` ‚Äî about section wrapper.
- `.about-block-label` ‚Äî label above about grid.
- `.about-grid` ‚Äî grid of cards.
- `.about-card` ‚Äî card-style link.

**Repository**

- `.repo-intro` ‚Äî intro paragraph under h1.
- `.repo-cell-mono` (or `.repo-num`, `.repo-shortname`, `.repo-date`) ‚Äî smaller mono in table cells.
- `.repo-subtitle` ‚Äî subtitle in title cell.
- `.repo-tags` ‚Äî tags cell.

**Groups**

- `.groups-intro` ‚Äî intro under h1.
- `.group-label-sub` or `.group-subtitle` ‚Äî italic subtitle in group label.

**About / static**

- Use **`.placeholder`** (or `.muted-italic`) for TBD and "No reviews completed yet."

---

## 8. Naming conventions used

- **Semantic where it helps:** `.hero`, `.script-direction`, `.related-label`, `.about-card`, `.section-head`.
- **Component + element:** `.hero-tagline`, `.hero-stats`, `.group-label-sub`.
- **Reuse:** `.placeholder` for all "not yet written" / TBD / muted italic text; `.link-mono-small` / `.link-mono-accent` for small caps links.

If you want flatter names (e.g. `.muted-italic` instead of `.placeholder`, or `.card-link` instead of `.about-card`), we can switch before applying in `build.py`.
